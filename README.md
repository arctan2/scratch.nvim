it also has a "compile" mode command to run commands in scatch buffer in terminal mode. Inspired by emacs compile mode.

# commands:

|cmd|description|
|---|-----------|
|`Scratch !<cmd?>`|new scratch buffer|
|`VScratch !<cmd?>`|new vertial split scratch buffer|
|`HScratch !<cmd?>`|new horizontal split scratch buffer|
|`FScratch !<cmd?>`|new floating scratch buffer|
|`FocusScratch !<cmd?>`|focus the floating scratch buffer|
|`Compile !<cmd?>`|new scratch buffer in terminal mode|
|`VCompile !<cmd?>`|new vertial split scratch buffer in terminal mode|
|`HCompile !<cmd?>`|new horizontal split scratch buffer in terminal mode|
